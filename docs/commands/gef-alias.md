gef> help gef-alias
GEF defined aliases
Syntax: gef-alias (set|show|do|unset)

List of gef-alias subcommands:

gef-alias do -- GEF do alias command

gef-alias show -- GEF show alias command
gef-alias unset -- GEF remove alias command
